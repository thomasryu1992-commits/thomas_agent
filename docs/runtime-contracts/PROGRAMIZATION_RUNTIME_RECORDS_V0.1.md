# Programization Runtime Records v0.1

**Status:** Review-Only Contract
**Runtime Activation:** Disabled

The Programization policy is represented by three auditable records:

```text
programization_observation.v0.1
↓
programization_pattern.v0.1
↓
programization_candidate.v0.1
```

A valid Observation excludes retries, revision cycles, duplicate replays, synthetic tests, fixtures, manual smoke tests, incomplete Tasks, and repeated identical input without an independent business event.

Ten valid independent Observations trigger Review only.

They do not create or activate a Program automatically.

A Candidate remains:

```text
candidate_only_pending_program_registry_and_permission_policy
```

Programization never expands Permission.

The Agent retains novel judgment, interpretation, strategy, and material exception handling.
